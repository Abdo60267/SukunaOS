# MDE Mockup (Qt/QML)

Este mockup demonstra a aparência inicial do `Malevolent Desktop Environment` usando Qt Quick (QML) e PySide6.

Pré-requisitos
- Python 3.8+
- PySide6 (instalar via `pip install PySide6`)

Executando

```bash
pip install PySide6
python3 mde/mockup/main.py
```

O mockup exibe uma janela com tema vermelho/negro/dourado, um dock e um painel de controle.

Notas
- Este é um protótipo visual; componentes reais do MDE serão implementados em C++/Qt e QML com serviços separados.
