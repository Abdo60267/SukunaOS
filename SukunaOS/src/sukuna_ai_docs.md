# SukunaAI — especificação básica

## Visão geral

SukunaAI é o assistente integrado do SukunaOS para ajudar usuários e desenvolvedores com comandos, automação, correção de erros e sugestões.

## Componentes

- `sukuna-ai-core` (backend local)
- `sukuna-ai-agent` (daemon que responde a solicitações do desktop)
- `sukuna-ai-frontend` (UI no MDE)

## Funcionalidades

- Sugestões de comandos de terminal.
- Ajuda contextual em apps e configurações.
- Revisão de erros e erros comuns.
- Assistência de instalação de pacotes.
- Automação de tarefas simples.

## Privacidade

- Processamento local por padrão.
- Dados não são enviados para nuvem sem consentimento.
- Logs e histórico gerenciados pelo usuário.
